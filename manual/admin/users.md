# Admin - Users

## Purpose
Admins manage user accounts for their assigned companies.

## Create a user
1. Go to **Settings â†’ Users** (or **Users** in the sidebar).
2. Click **Create**.
3. Fill in the user details (name, email, status).
4. Assign the appropriate role.
5. Save.

## Edit or deactivate users
- Use **Edit** to update user details.
- Change status to disable access when needed.

## Access rules
- Admins can only manage users within their assigned companies.
- Admins cannot create Super Admin accounts.

## Tips
- Keep roles accurate to prevent access issues.
- Use consistent naming for users.
