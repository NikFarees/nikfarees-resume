# Vendor Admin - Pricing and Catalog

## Purpose
Pricing tables and catalog items guide how services are billed. Vendor Admins typically have **read-only** access to this data.

## Where to view pricing
- **Pricing Table → Pricing Tables**
- **Pricing Table → Catalog Items**

## How to use pricing
- Reference pricing while creating or reviewing service orders.
- If pricing looks incorrect, contact a Super Admin to review.
