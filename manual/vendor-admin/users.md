# Vendor Admin - Users

## Purpose
Vendor Admins manage **vendor staff** accounts for their vendor company.

## Create a vendor staff user
1. Go to **Settings â†’ Users** (or **Users** in the sidebar).
2. Click **Create**.
3. Fill in the user details (name, email, status).
4. Assign the role **Vendor Staff**.
5. Save.

## Edit or deactivate users
- Use the **Edit** action to update user details.
- Change status to disable access when needed.

## Access rules
- Vendor Admins can only see users linked to their vendor company.
- Vendor Admins cannot create Super Admin or Admin accounts.

## Tips
- Use consistent naming for staff accounts.
- Keep roles accurate to avoid access issues.
