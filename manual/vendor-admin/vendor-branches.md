# Vendor Admin - Vendor Branches

## Purpose
Vendor branches represent your locations or operational hubs. Branches are used to route work and organize staff.

## Create a branch
1. Go to **Service Partners â†’ Vendor Branches**.
2. Click **Create**.
3. Enter branch details (name, address, contact info).
4. Save.

## Assign staff to branches
- After creating a branch, add staff members to the branch from the **Users** section.
- Only staff assigned to a branch can be scheduled for work at that branch.

## Tips
- Keep branch names consistent with internal naming.
- Use a single primary contact for each branch.
